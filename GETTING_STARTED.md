# Learn R From Scratch – Getting Started

Thank you for testing **Learn R From Scratch**!

This course is currently in beta. Your feedback will help improve both the content and the learning experience.

---

## 1. Install the Required Software

Before starting the course, please install:

- **R** (latest version): https://cran.r-project.org/
- **RStudio Desktop** (free version): https://posit.co/download/rstudio-desktop/

Both applications are available for Windows, macOS and Linux.

---

## 2. Download the Course

You have two options.

### Option 1 – Clone the GitHub repository

If you are familiar with Git, clone the repository:

```bash
git clone https://github.com/duretteba/Learn-R-From-Scratch.git
```

### Option 2 – Download the ZIP archive

If you are not familiar with Git:

1. Go to the GitHub repository.
2. Click the green **Code** button.
3. Click **Download ZIP**.
4. Extract the ZIP archive to a folder on your computer.

---

## 3. Open the Course in RStudio

Start **RStudio**.

Open the **Learn-R-From-Scratch** folder on your computer.

You should see a project structure similar to this:

```text
Learn-R-From-Scratch/
│
├── modules/
│   ├── images/
│   ├── module1-introduction.qmd
│   ├── module2-data-visualization.qmd
│   ├── module3-data-manipulation.qmd
│   ├── module4-working-with-tables.qmd
│   └── module5-loading-your-own-data.qmd
│
├── datasets/
├── README.md
└── ...
```

---

## 4. Start the First Module

In the **Files** pane of RStudio:

1. Open the **modules** folder.
2. Double-click **module1-introduction.qmd**.

---

## 5. Run the Code

The course is interactive.

Each module contains executable R code.

Run each code chunk by clicking the **Run** button in RStudio or by using the keyboard shortcut:

- **Ctrl + Enter** on Windows/Linux
- **Cmd + Enter** on macOS

Whenever you are asked to complete an exercise, try to solve it before moving on.

Some modules require additional R packages such as **ggplot2**, **tidyverse** or **readxl**.

The first time you encounter one of these packages, simply execute the installation command provided in the module.
