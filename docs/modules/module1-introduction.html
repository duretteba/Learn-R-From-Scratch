3.1 1 - Performing Basic Arithmetic Operations with R

At a minimum, R can be used as a powerful calculator. Addition, subtraction, multiplication, and division are written as follows (click the green arrow in the upper-left corner of each cell to display the result):

Addition

# Addition
1+1
[1] 2

Subtraction

# Subtraction
1-1
[1] 0

Multiplication

# Multiplication
2*2
[1] 4

Division

# Division
4/2
[1] 2

For exponentiation, use the ^ symbol.

# Power
2^3
[1] 8

For square roots, you can either use the sqrt function or raise a number to the power of 1/2.

# Square root using sqrt
sqrt(16)
[1] 4
# Square root using exponentiation
16^(1/2)
[1] 4

R also provides all standard trigonometric functions:

# Cosine
cos(20)
[1] 0.4080821
# Sine
sin(10)
[1] -0.5440211
# Tangent
tan(10)
[1] 0.6483608
# Arctangent
atan(10)
[1] 1.471128

And logarithmic functions:

# Natural logarithm (base e)
log(1)
[1] 0
# Base-10 logarithm using log10
log10(10)
[1] 1
# Base-10 logarithm using log
log(10, base = 10)
[1] 1

That concludes a first overview of the most commonly used functions.

3.2 2 - Viewing a Data Table in R

R includes several built-in datasets that are ready to use. Some of the most popular are:

  1. mtcars: data on cars

  2. iris: data on flowers

  3. ToothGrowth: data on tooth growth

  4. PlantGrowth: data on plant growth

  5. USArrests: data on crime statistics

To learn more about these datasets, you can use the help function, which opens documentation in a separate RStudio tab. The data() function provides access to all built-in datasets.

help("ToothGrowth")

To access a dataset, simply type its name. However, if you do that (go ahead and try!), R will display the entire dataset by default. That can be difficult to read. Here are a few useful functions for exploring a dataset.

The dim function returns the dimensions of the table. Here, there are 150 rows and 5 columns.

# dim function
dim(iris)
[1] 150   5

The names function returns the column names:

# names function
names(iris)
[1] "Sepal.Length" "Sepal.Width"  "Petal.Length" "Petal.Width"  "Species"

Finally, the head function provides a preview of the first rows of the dataset:

head(iris)

3.3 3 - How to Perform Basic Arithmetic Operations on a Data Table?

Now let’s get to the heart of the matter. How can you use R to manipulate data and perform basic operations?

The first of the four essential symbols you need to know in R is the “$” symbol (money, money, money!).

The $ symbol is used to select a column from a data frame. For example, if I want to retrieve the “Sepal.Length” column from the “iris” dataset, I would write:
